# Video Files

## Required Video

Place your truck highway video file in this directory with the following name:
- `truck-highway.mp4` (primary format)
- `truck-highway.webm` (optional, for better browser compatibility)

### Video Recommendations

For optimal performance and clarity:
- **Resolution**: 1920x1080 (Full HD) or higher
- **Format**: MP4 (H.264 codec) and/or WebM (VP9 codec)
- **File Size**: Keep under 10-15 MB for faster loading
- **Duration**: 10-30 seconds (it will loop automatically)
- **Content**: A running truck on a highway, preferably from a side or front angle
- **Quality**: High quality, well-lit footage for maximum clarity

### Where to Find Videos

You can find free stock videos at:
- Pexels Videos (https://www.pexels.com/videos/)
- Pixabay Videos (https://pixabay.com/videos/)
- Videvo (https://www.videvo.net/)

Search terms: "truck highway", "semi truck driving", "transport truck road"

### Compression Tips

To optimize your video file size:
- Use HandBrake (https://handbrake.fr/) or FFmpeg
- Target bitrate: 2-4 Mbps for web use
- Remove audio track if not needed (our video is muted)
